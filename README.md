## üöÄ Getting Started

### Prerequisites

- Python 3.6 or higher
- Git (for cloning the repository)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/file-organizer.git
cd file-organizer
```

2. Run the script:

```bash
python main.py
```
